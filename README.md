# Pedrero Social Media - Sitio Web Oficial

Sitio web profesional de la agencia de social media marketing Pedrero Social Media.

## 🚀 Características

- Diseño minimalista orgánico con paleta terrosa y acentos dorados
- Video de fondo animado en hero principal
- Líneas SVG animadas para efectos visuales dinámicos
- Portfolio con filtros por categoría (Reels, Eventos, Colaboraciones, Contenido)
- Carrusel de testimonios de clientes reales
- Sección de servicios completa
- Formulario de contacto
- Páginas legales (Aviso Legal, Política de Privacidad)
- 100% responsive y optimizado para SEO
- Animaciones al hacer scroll con Intersection Observer

## 🛠️ Stack Tecnológico

- **Frontend**: React 19 + TypeScript
- **Styling**: Tailwind CSS 4
- **UI Components**: shadcn/ui
- **Routing**: Wouter
- **Build Tool**: Vite
- **Deployment**: Manus Platform

## 📦 Instalación

```bash
# Instalar dependencias
pnpm install

# Iniciar servidor de desarrollo
pnpm dev

# Build para producción
pnpm build
```

## 🚀 Despliegue

### Despliegue en Plesk

Para desplegar en un servidor Plesk, consulta la guía completa en [DEPLOY_PLESK.md](./DEPLOY_PLESK.md).

**Método rápido**:

```bash
# Ejecutar script de build
./deploy.sh

# Subir el contenido de dist/public/ a httpdocs/ en Plesk
```

### Otras Plataformas

- **Vercel**: Conecta el repositorio desde vercel.com
- **Netlify**: Conecta el repositorio desde netlify.com
- **Manus**: Usa el botón "Publish" en la UI

## 🎨 Filosofía de Diseño

**Minimalismo Orgánico con Textura**
- Paleta terrosa (beige, terracota, carbón) con acentos dorados
- Tipografía: Fraunces (títulos), Literata (cuerpo), DM Sans (botones)
- Bordes redondeados orgánicos variables
- Animaciones suaves con curvas naturales
- Espaciado generoso y fluido
